# Microcontroller Carrier PCBs

Custom carrier boards for popular microcontrollers.

## Available Boards

- **Arduino Nano Carrier**
- **ESP32 Carrier**  
- **NodeMCU Carrier**
- **Universal Carrier**

All boards provide expanded pin access with extra power and ground terminals through T-block connectors.

## Quick Start

1. Install Proteus design files and fine-tune the PCB to your needs
2. Insert microcontroller into female headers
3. Connect power via multiple VIN/GND terminals  
4. Wire peripherals to screw terminals
5. Upload code via native USB/programming interface
6. Happy PCB population!

Eliminates direct soldering to development boards while providing secure screw terminal connections for all I/O and power lines.

**Note**: Verify voltage compatibility before use.
